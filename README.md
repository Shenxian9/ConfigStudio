# ConfigStudio

ConfigStudio 是一个基于 **Qt Widgets + Qwt** 的工业组态/可视化编辑项目，支持在画布中拖拽组件、编辑属性、绑定变量并进行运行时数据展示。

## 主要功能

- **可视化画布编辑**：支持在设计区添加/选择组件并调整属性。
- **属性面板编辑**：在右侧属性表中直接修改组件参数。
- **组件库**：包含图表、仪表、指示灯、文本、开关、滑块等常见组件。
- **数据绑定**：组件可通过 `varId` 与变量模型绑定，配合仿真器进行数据驱动更新。
- **运行预览**：支持切换到运行窗口/全屏模式查看效果。

## 项目结构（简要）

- `components/`：各类可视化组件实现（如 `indicator`、`plot`、`dial` 等）。
- `ui/`：主界面与属性编辑交互逻辑。
- `model/`：变量模型与数据结构。
- `runtime/`：运行时绑定与仿真逻辑。
- `canvas/`：画布与图元交互基础能力。

## 构建与运行（Qt/qmake）

```bash
qmake ConfigStudio.pro
make -j
./ConfigStudio
```

> 说明：需要本机已安装 Qt 开发环境（含 qmake）以及项目依赖库（如 Qwt）。
